# Описание типов стен в лабиринте

Наше 3D приложение лабиринта содержит 6 различных типов стен с заметными цветовыми и текстурными отличиями.
Каждый тип стены выполняет определенную функцию в структуре лабиринта:

## Тип 0: Внешние стены (с темным оттенком)
- Позиция в текстурном атласе: Верхний левый угол (0,0)
- Текстурный эффект: Стандартная текстура без искажений
- Цветовой эффект: Темный оттенок (яркость уменьшена на 20%)
- Используется для всех внешних стен лабиринта (периметр)
- Создает четкую границу лабиринта

## Тип 1: Вертикальные внутренние стены (красноватый оттенок)
- Позиция в текстурном атласе: Верхний центр (1,0)
- Текстурный эффект: Вертикальные полосы (повторение по горизонтали)
- Цветовой эффект: Красноватый оттенок (красный канал усилен на 40%)
- Используется для стен, соединяющихся только по вертикали
- Помогает в визуальной ориентации при передвижении с севера на юг

## Тип 2: Горизонтальные внутренние стены (синеватый оттенок)
- Позиция в текстурном атласе: Верхний правый угол (2,0)
- Текстурный эффект: Горизонтальные полосы (повторение по вертикали) 
- Цветовой эффект: Синеватый оттенок (синий канал усилен на 40%)
- Используется для стен, соединяющихся только по горизонтали
- Четко отличается от вертикальных стен, помогая в навигации запад-восток

## Тип 3: Перекрестки (зеленоватый оттенок)
- Позиция в текстурном атласе: Нижний левый угол (0,1)
- Текстурный эффект: Клетчатый узор (шахматная сетка)
- Цветовой эффект: Зеленоватый оттенок (зеленый канал усилен на 40%)
- Используется в местах пересечения коридоров
- Ключевые точки для навигации, обозначающие места принятия решений

## Тип 4: Концы коридоров (желтоватый оттенок)
- Позиция в текстурном атласе: Нижний центр (1,1)
- Текстурный эффект: Диагональные полосы
- Цветовой эффект: Желтоватый оттенок (красный и зеленый каналы усилены на 40%)
- Используется для стен на концах коридоров, не соединяющихся с другими стенами
- Обозначает тупики и потенциальные места для размещения объектов

## Тип 5: Декоративные стены (фиолетовый оттенок)
- Позиция в текстурном атласе: Нижний правый угол (2,1)
- Текстурный эффект: Радиальный узор (концентрические круги)
- Цветовой эффект: Фиолетовый оттенок (красный и синий каналы усилены на 40%)
- Используется для особых стен, встречающихся редко (каждая 7-я стена)
- Может служить в качестве ориентиров или обозначать особые локации

Комбинация текстурных узоров и цветовых оттенков создает визуально разнообразную и легко различимую среду, 
которая не только улучшает эстетический вид лабиринта, но и помогает в навигации, делая
исследование лабиринта более интуитивным и интересным.
